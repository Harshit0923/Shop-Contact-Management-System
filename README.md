# Shop Contact Management System (C)

This project is a simple C-based application designed for managing contacts of customers and suppliers in a shop.

## Features
- Add new contact
- Display all contacts
- Search contact
- Delete contact
- Filter contacts by type (Customer/Supplier)

## Data Structure Used
Doubly Linked List

## Files Included
- contact.c → Source code
- contact.exe → Executable file (ready to run)

## How to Run

### Option 1: Run directly (Recommended)
1. Open the folder
2. Double-click `contact.exe`

No compiler required

---

### Option 2: Run using compiler
1. Open terminal in the folder
2. Compile:
   gcc contact.c -o contact
3. Run:
   ./contact

## System Requirements
- Windows OS (for .exe file)
- GCC compiler (only needed if running source code)

## Real-Life Use
This system helps shop owners manage and organize contact details of customers and suppliers efficiently.
